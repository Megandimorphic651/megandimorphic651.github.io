## 📖 Overview

SpookiUI serves as a configuration tool for the Ghostly terminal. It provides a visual interface for users who want to change their terminal settings without editing text files. Ghostly terminal offers high performance for command-line users, but its configuration often requires manual input. SpookiUI simplifies this process. You select your preferences through menus. The application saves these changes to your configuration profile automatically.

## 💻 System Requirements

Your computer must meet these basic standards to run SpookiUI:

*   Processor: Standard dual-core processor or better.

## ⚙️ How to use the interface

SpookiUI organizes settings into logical categories. When you launch the application, you see a main menu with several tabs.

### Terminal Colors
This tab lets you adjust the color palette for your terminal. You can pick from pre-made themes or choose custom colors for your background and text. Click on a color box to open a color picker. After you select a color, click Save to apply the change.

### Font and Text
The text settings allow you to change the font style and size. SpookiUI lists every font currently installed on your computer. Select a font from the list, choose your desired size, and check how it looks in the preview window at the bottom of the screen.

### Cursor and Input
You can change the shape and behavior of your cursor here. Select between blocks, vertical bars, or underlines. You can also turn off the blinking effect if it distracts you.

### Profile Management
If you use multiple setups for different projects, use this section to save them as distinct profiles. Give your profile a name, adjust your settings, and save. You can switch between these profiles whenever you need a different environment.

## 🔧 Troubleshooting Common Issues

If you encounter problems, check these solutions:

*   **Application does not open:** Ensure you have administrator rights on your computer. Right-click the icon and choose "Run as administrator."
*   **Settings do not change:** Confirm that Ghostly terminal is closed before you save new settings. The terminal often locks files while it runs, which prevents SpookiUI from writing changes.
*   **Error messages:** If you see a notification about a missing file, restart your computer and try opening the application again. If the issue remains, reinstall the program using the link above.

## 🛡️ Privacy and Safety

SpookiUI does not connect to the internet. It only accesses the configuration files stored on your local hard drive. The application does not collect your data, send usage statistics, or share your terminal settings with third parties. Your personal information stays on your device at all times.

## 📝 Frequently Asked Questions

**Does SpookiUI work with other terminals?**
No, this version works specifically with the Ghostly terminal.

**Will this app make my terminal slower?**
No. SpookiUI only writes to a text file once you click the save button. It does not run in the background while you use your terminal.

**Can I undo my changes?**
Yes. You can revert to the default settings at any time by clicking the "Restore Defaults" button inside the settings menu.

**Where does SpookiUI keep my files?**
The app saves your settings in your personal user folder under a subfolder named GhostlyConfig.
